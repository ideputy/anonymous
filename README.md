# Examples of Questions and Moodle XML Generation
This repository provides example assessment items and a Jupyter notebook for generating questions in **Moodle XML** format, with **multi-language support** and generators for **supporting materials**.

Interactive examples are available at the **[demo site](https://ideputy.gitbook.io/test)** (anonymized)

## Examples
- **Item 6** — character substitution
- **Item 7** — invisible space
- **Items 8–9** — hindered True/False
- **Item 12** — vague introduction with hidden prompts
- **Item 14** — wavy text-to-image question
## Notebook
- `question_generator.ipynb` — generates Moodle XML questions and supporting materials
